# Capstone Financial NLP

금융 자연어처리(NLP) 캡스톤 프로젝트입니다.

## 프로젝트 구조

```
capstone_financial_nlp/
├─ app/                    # 프로젝트 주요 코드
├─ data/                   # 데이터셋 및 처리 결과
│  ├─ metadata/            # 데이터 설명 또는 메타데이터
│  ├─ processed/           # 전처리된 데이터
│  ├─ raw/                 # 원본 데이터
│  ├─ unified/             # 통합 및 정리된 데이터
│  ├─ dataset_samples.txt  # 데이터 샘플 설명
│  ├─ dataset_summary.csv  # 데이터셋 요약 정보
│  ├─ financial_sentiment.csv  # 금융 감정 데이터
│  └─ fiqa_raw.csv         # FiQA 원본 데이터
├─ results/                # 실험 결과
│  ├─ figures/             # 시각화 그래프
│  └─ reports/             # 분석 보고서
├─ README.md
├─ .gitignore
└─ LICENSE
```

## 시작하기

```bash
pip install -r requirements.txt
```
